## Usage

```
terraform init
terraform plan
terraform apply
```

## Outputs

| Name | Description |
|------|-------------|
| user\_arn | ARN of the IAM user |
| user\_name | Name of the IAM user |
| group\_arn | ARN of the IAM group |
| group\_name | Name of the IAM group |
| role\_arn | ARN of the IAM role |
| role\_name | Name of the IAM role |
